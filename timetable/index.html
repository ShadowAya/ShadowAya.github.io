Dynamic Timetable version 1.0

A timetable inside a notification. Fetches data daily and updates automatically. Detects off-days.
Built-in buttons for easy access to commute etc. Currently works with EduPage only!



Some technical skill is needed to set-up, but following the installation is straight-forward.

Requirements

    • Apps
    • Tasker
    • Termux
    • Termux:Tasker
    • AutoNotification
    • Total Commander (recommended)
  • Tasker Profile
  • Python Script
  • Login JSON
